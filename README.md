# Number Guessing Game

A simple web-based number guessing game that prompts users to guess a randomly generated number between 1 and 10. This project is built using HTML, CSS, and JavaScript.

## Overview

In this game, users are tasked with guessing a random number between 1 and 10. After each guess, feedback is given to help the player find the correct number with hints if their guess is too high or too low.

## Features

- Random number generation from 1 to 10.
- User input field to submit guesses.
- Feedback after each guess, indicating if the number is higher or lower than the guess.
- Display of the number of attempts made when the correct answer is guessed.

## How to Play

## Code Structure

- **HTML**: Sets up the structure of the webpage, including a heading, instructions, input field, and button for submitting guesses.
- **CSS**: Basic styling for layout and text to ensure the game looks clean and centered.
- **JavaScript**: Core logic of the game.
  - Generates a random number on page load.
  - Tracks the number of guesses.
  - Provides feedback to the user about each guess.
